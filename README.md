#  DecodeLabs Internship Project 1: Data Cleaning Preparation

## Project Overview
This project is part of my internship at DecodeLabs.  
The goal is to clean a raw dataset by handling missing values, duplicates, and incorrect data formats so the data becomes reliable for analysis.

## Dataset Description
The dataset contains order information such as:
- OrderID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Item in Cart
- Coupon Code
- Referral Source
- Total Price

## Objectives
- Identify missing values.
- Remove duplicate records.
- Correct formatting issues.
- Make the dataset ready for analysis.

## Data Cleaning Steps
- Checked for missing or null values.
- Removed duplicate rows and duplicate OrderIDs.
- Standardized date format.
- Verified numeric fields like quantity and unit price.
- Checked text fields for consistency.
- Confirmed total price values where needed.

## Tools Used
- Microsoft Excel / Power BI / Python
- GitHub for version control

## Expected Analysis
After cleaning, the dataset can be used to analyze:
- Best-selling products
- Revenue trends
- Customer behavior
- Payment method preferences
- Order status distribution
- Referral source performance

## Conclusion
This project helped me practice data cleaning and prepare a raw dataset for further analysis. It also improved my understanding on how to identify missing values and making dates formatting.
